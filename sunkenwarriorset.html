The Sunken Warrior Set is a relic armor set that consists of the Sunken Warrior Helm, Sunken Warrior Chestplate, and Sunken Warrior Legs. It is an ornate armor set with gilded trim and colored metallic plates. These plates are recolorable.


Inspiration

The Sunken Warrior Set was inspired by corresponding Sunken Warrior Set from Arcane Odyssey.
